btools 旨在打造更好、更实用的 JavaScript 工具库，该库是一套在实际项目中使用频率较高的模块集合。

## 使用方法

安装 **btools** 后直接引用。

```bash
npm i btools --save
```

### 全部引入

```javascript
import btools from 'btools';
```

### 单模块引入

```javascript
import { toType } from 'btools';
```

### 浏览器中使用

```javascript
<script src='./btools.js'></script>
```
